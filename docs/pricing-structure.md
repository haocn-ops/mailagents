# Pricing Structure (Draft)

This draft is optimized for early adoption and clear upgrade paths.

## Free Tier

- 1 active lease at a time
- 100 inbound messages / month
- 100 outbound messages / month
- Shared inbox domain (`inbox.mailagents.net`)
- Community support

## Starter ($29 / month)

- 3 concurrent leases
- 5,000 inbound messages / month
- 5,000 outbound messages / month
- Webhooks
- Standard support

## Pro ($149 / month)

- 20 concurrent leases
- 50,000 inbound messages / month
- 50,000 outbound messages / month
- Custom domains
- Priority support

## Enterprise (Custom)

- Unlimited leases and volume tiers
- Dedicated IP / deliverability support
- SLA and onboarding

## Usage-Based Overage

- Over-limit requests billed via existing HMAC proof flow.
- Display current usage and projected cost in the user app.
